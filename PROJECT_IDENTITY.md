# Project Identity Policy

The name "PhobosChemistryPathways" represents the official maintained release.

Allowed:
- Fork and modify
- Redistribute modified versions
- Create addons and compatibility patches

Required:
- Clearly label forks as unofficial
- Provide attribution

Not allowed:
- Reupload unchanged under same name
- Imply official status
- Cause authorship confusion

Official distribution: Steam Workshop page maintained by original author.
