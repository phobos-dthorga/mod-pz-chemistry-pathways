# PhobosChemistryPathways — Source Art Assets

1024x1024 PNG source images for item icons. These are NOT game-required files — the game loads
only the 128x128 `Item_PCP_*.png` files found in the `main` branch.

These images are used for:
- Steam Workshop preview screenshots
- Discord community communication
- Documentation and promotional materials
- Source material for generating new Item_PCP inventory icons

See the [main branch](../../tree/main) for the mod itself.
